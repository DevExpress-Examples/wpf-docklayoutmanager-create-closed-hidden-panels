# WPF Dock Layout Manager - Create Closed (Hidden) Panels

This example creates closed (hidden) panels. You can use the [DockLayoutManager.ClosedPanels](https://docs.devexpress.com/WPF/DevExpress.Xpf.Docking.DockLayoutManager.ClosedPanels) collection to access closed panels. In the example, the built-in Closed Panels bar is visible (itsÂ [DockLayoutManager.ClosedPanelsBarVisibility](https://docs.devexpress.com/WPF/DevExpress.Xpf.Docking.DockLayoutManager.ClosedPanelsBarVisibility) property is `Auto`).
